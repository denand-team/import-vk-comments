Плагин импортирует комментарии из виджета комментариев ВК в WordPress.

== Description ==

Плагин создан для тех, кто планирует перенести свои комментарии из виджета от соцсети Вконтакте в базу данных сайта Wordpress.

Для импорта понадобится сервисный ключ доступа и ID виджета комментариев. Подробнее о их получении будет описано на главной странице плагина.

На данный момент комментарии можно получить только к типу постов "Записи". После импорта они появятся во вкладке "Ожидающие модерации".

= Отладка =

Так как получение комментариев идет с задержками, что-то может пойти не так, к примеру со стороны VK API. Необходим доступ сайта к серверам Вконтакте.

В Console.log передаются подробные сведения о состоянии задач.


== Installation ==

1. Распакуйте содержимое `import-vk-comments.zip` в папку `/wp-content/plugins/` или воспользуйтесь поиском по плагинам.
2. После установки во вкладке 'Плагины' Вашего WordPress сайта нажмите Активировать.
3. Пропишите сервисные ключи и ID виджета, затем жмите "Сохранить".
4. Нажмите кнопку импортировать и дождитесь 100%.

== Frequently Asked Questions ==

= Куда попадут комментарии =

Они будут во вкладке "Ожидают модерации" вашего Wordpress сайта.

= Что за имена User****, Anonym*** =

Пользователи, имена которых уже не получить (помеченные DELETED) в VK, автоматически переименовываются в данном стиле.
